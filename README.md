# 我的个人网站

这是一个用 HTML、CSS 和 JavaScript 搭建的个人网站，用于存储图片和编辑文字。

## 功能特点

- 📷 图片上传和展示
- 📝 在线文字编辑和发布  
- 📱 响应式设计，支持手机访问
- 🎨 现代化界面设计

## 访问网站

点击这里访问：[https://你的用户名.github.io/Hao](https://你的用户名.github.io/Hao)

## 使用方法

1. 在图片库区域点击或拖拽上传图片
2. 在随笔区域写下文字并发布
3. 所有内容会保存在浏览器本地存储中

## 技术栈

- HTML5
- CSS3  
- JavaScript (ES6+)
- GitHub Pages 部署
